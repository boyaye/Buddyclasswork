CSS positioning

static positioning

Element is in normal flow.This is a normal value

Relative Postion

Element in normal flow but it offset base on provided value.

The values top,right,left,and bottom can be offset element.

Absolute positioning

The element is removed from a normal flow.

its final position is relative to the positioned parent or ancestors

and is determined by the value, of top , bottom, right,and left.

Fixed positioning

The element is removed from the normal document flow.

The final position is relative to the viewport and determined by the value of top,right,left.bottom.
